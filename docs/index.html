🚀 Quick Start

1. Install Dependencies

pip install -r requirements.txt

2. Configure Credentials

# Edit .env file with your testnet credentials BINANCE_API_KEY=your_key_here BINANCE_API_SECRET=your_secret_here

3. Run the Bot

# Interactive mode python trading_bot/cli.py # Or command line python trading_bot/cli.py place-order BTCUSDT BUY MARKET 0.001

📖 Available Commands

python cli.py place-order BTCUSDT BUY MARKET 0.001 # Market order python cli.py place-order BTCUSDT SELL LIMIT 0.001 --price 50000 # Limit order python cli.py balance # Check balance python cli.py status BTCUSDT 12345 # Check order status python cli.py cancel BTCUSDT 12345 # Cancel order python cli.py menu # Interactive mode
